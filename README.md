# **🚀 Belajar Dart & Flutter | Tugas & Praktikum**

Selamat datang di repositori pembelajaran **Dart dan Flutter** saya\!

Repositori ini berfungsi sebagai jurnal perjalanan belajar, tempat penyimpanan tugas praktikum, dan *showcase* proyek mini yang akan terus diperbarui setiap minggunya (Week 1, Week 2, Week 3, dan seterusnya).

## **🛠️ Tech Stack**

* **Bahasa Pemrograman:** [Dart](https://dart.dev/)  
* **Framework UI:** [Flutter](https://flutter.dev/)  
* **IDE / Code Editor:** Visual Studio Code / Android Studio (sesuaikan dengan yang kamu pakai)

## **📂 Struktur Repositori**

Repositori ini dibagi berdasarkan minggu pertemuan / modul pembelajaran. Setiap folder akan berisi source code, latihan, dan *challenge* terkait topik pada minggu tersebut.

## **💻 Cara Menjalankan Program**

### **Untuk File Dart murni (CLI)**

Jika kamu ingin menjalankan file Dart sederhana (seperti tugas Week 2), buka terminal, arahkan ke folder yang sesuai, dan jalankan perintah:

\# Contoh menjalankan tugas Week 2  
cd Week\_2  
dart run latihan\_1.dart

### **Untuk Proyek Flutter (Nantinya)**

Jika folder tersebut adalah proyek aplikasi Flutter, gunakan perintah berikut di terminal:

\# Masuk ke direktori proyek Flutter  
cd nama\_folder\_proyek

\# Ambil \*dependencies\* (paket) yang dibutuhkan  
flutter pub get

\# Jalankan aplikasi di emulator atau \*device\* yang terhubung  
flutter run

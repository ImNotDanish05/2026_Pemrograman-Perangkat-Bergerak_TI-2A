Tentu, ini resume tugas-tugas yang harus kamu selesaikan berdasarkan *jobsheet* praktikum Dart tersebut:

### 1. Guided Practice (Praktik Terpandu)

* Membuat program Dart yang menyimpan informasi pribadi, kemudian mencetaknya ke konsol. Informasi tersebut meliputi:


* Nama 


* ID Mahasiswa (NIM) 


* Jurusan 





### 2. Exercises (Latihan Utama)

* 
**Exercise 1 - Simple Calculator:** Membuat program kalkulator sederhana yang dapat menghitung penjumlahan, pengurangan, perkalian, dan pembagian dari dua angka.


* 
**Exercise 2 - Even or Odd:** Membuat program untuk mengecek apakah sebuah angka merupakan bilangan genap atau ganjil.


* 
**Exercise 3 - Multiplication Table:** Menampilkan tabel perkalian untuk angka 5 (dari 1 hingga 10) ke layar.



### 3. Challenge Task (Tugas Opsional)

* Membuat sebuah fungsi untuk menghitung luas persegi panjang.


* Fungsi tersebut menggunakan rumus: Luas = panjang x lebar.



### 4. Reflection Questions (Pertanyaan Refleksi)

* Menjawab empat pertanyaan pemahaman teori berikut:
1. Apa itu Dart dan mengapa digunakan dalam Flutter? 


2. Apa perbedaan antara tipe data var dan int? 


3. Apa tujuan dari penggunaan fungsi (function)? 


4. Kapan perulangan (loops) sebaiknya digunakan? 





### 5. Ketentuan Pengumpulan (Submission)

* Kamu perlu mengumpulkan tiga hal:
1. File *source code* dari program Dart (.dart).


2. 
*Screenshot* hasil eksekusi program.


3. Penjelasan singkat mengenai masing-masing tugas *Exercise*.

**Format File:** Semua tugas tersebut disatukan dan dikumpulkan dalam format `.zip` dengan penamaan file: `NIM_Name_DartLab2.zip`.
